# Bingo_Sofka
<h2>Bingo El gran buda-sofka</h2>

Este proyecto es un Bingo Virtual desarrollado para la empresa "El Gran Buda". El Bingo Virtual permitirá a los usuarios registrarse y jugar a un juego de bingo en línea.

Este proyecto  de bingo virtual es desarrollado con  tecnologias como:

<ul>
<li>Java con Spring Boot</li>
<li>MySql</li>
<li>Node.js</li>
<li>MongoDB</li>
<li>Vite+React</li>
</ul>
